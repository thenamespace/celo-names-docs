---
title: "How to register"
description: "Register a Celo Name in a few steps"
---

1. Connect your wallet to Celo.
2. Search for an available name.
3. Follow the registration flow to pay fees and set a resolver.
4. Optionally set your primary name (reverse record).

<Note>
For UX reference, see Base Basenames tutorials: [Wagmi](https://docs.base.org/base-account/basenames/basenames-wagmi-tutorial) and [OnchainKit](https://docs.base.org/base-account/basenames/basenames-onchainkit-tutorial).
</Note>
